Lesson 7: TRANSFORMING EXPONENTIAL/
LOGARITHMIC FORMS WITH
EACH OTHER


From Exponential to Logarithmic: An exponential equation like a^b = c can be rewritten as the logarithmic form log_a(c) = b. This expresses the exponent b as the logarithm of c with base a.
From Logarithmic to Exponential: A logarithmic equation like log_a(c) = b can be rewritten as the exponential form a^b = c. This expresses c as the result of raising a to the power of b.
In short, exponential and logarithmic functions are inverses of each other, and they can be converted back and forth using these rules.




Lesson 8: LAWS OF LOGARITHMS


Laws of logarithms are rules that can be applied to simplify and solve complicated logarithmic equations.

Product Rule:
log_a(x * y) = log_a(x) + log_a(y)

Quotient Rule:
log_a(x / y) = log_a(x) - log_a(y)
Power Rule:
log_a(x^n) = n * log_a(x)

Change of Base Formula:
log_a(b) = log_c(b) / log_c(a)
(Where c is any positive number, usually 10 or e.)

Logarithm of 1:
log_a(1) = 0
(For any base a > 0.)

Logarithm of the Base:
log_a(a) = 1
(For any base a > 0.)

Logarithm of a Reciprocal:
log_a(1/x) = -log_a(x)





Lesson 9: INVERSE OF EXPONENTIAL/
LOGARITHMIC FUNCTIONS


The inverse of exponential and logarithmic functions reverses their operations:

Exponential Function:
y = b^x Its inverse is the logarithmic function: f^(-1)(x) = log_b(x), which solves for the exponent x given the base b.

Logarithmic Function:
y = log_b(x) Its inverse is the exponential function: f^(-1)(x) = b^x, which calculates the value y given the base b raised to the power of x.

Exponential Function:
y = b^x
x = b^y
log_b(x) = y
f^(-1)(x) = log_b(x)

Logarithmic Function:
y = log_b(x)
x = log_b(y)
b^x = y
f^(-1)(x) = b^x

Lesson 13: CONTINUOUS COMPOUNDING


Continuous compounding is the process of calculating interest where it is compounded infinitely many times within a given time period. Instead of compounding at intervals (like daily, monthly, or yearly), interest is added at every possible moment, leading to the maximum possible return.

Formula:
A = P * e^(rt)   or   F = Pe^(rt)

Where:
A = future value
P = principal amount
r = annual interest rate (in decimal form)
t = time in years
e = mathematical constant (approximately 2.718)

Continuous compounding results in slightly higher returns compared to standard compounding at fixed intervals.




Lesson 14: PROPOSITION


- A declarative statement that is either true (T) or false (F) but it won't be both.

4 types of sentences:
- declarative        - interogative
- imperative         - exclamatory

Simple & Compound Proposition
Simple - a proposition that can't be broken down any further into components of proposition
Compound - a proposition that is formed from combining two or more simple proposition using logic connectives

Negation (NOT)
Symbol: ¬
Formula: ¬P
Description: True if P is false.

Conjunction (AND)
Symbol: ∧
Formula: P ∧ Q
Description: True if both P and Q are true.

Disjunction (OR)
Symbol: ∨
Formula: P ∨ Q
Description: True if at least one of P or Q is true.

Conditional (IF... THEN...)
Symbol: →
Formula: P → Q
Description: True if whenever P is true, Q is also true.

Bi-conditional (IF AND ONLY IF)
Symbol: ↔
Formula: P ↔ Q
Description: True if P and Q are either both true or both false.




Lesson 15: COMPOUND PROPOSITION


A Compound Proposition is a logical statement formed by combining two or more simple propositions (or statements) using logical connectives. These propositions can be connected through operations like conjunction, disjunction, negation, conditional, and biconditional.
